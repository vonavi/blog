title:  "Схема нахождения слабейшего предусловия для цикла"
---

Рассмотрим цикл, эквивалентный функции `strlen`:
```C
while (a[i] != '\0') { i++ };
```
Постусловие после выхода из `while` цикла $$S$$ обозначим через $$R$$, а слабейшее предусловие как $$Q(i) = wp(S, R)$$.

Наша задача -- найти инвариант цикла $$S$$. В простейшем случае достаточно выполнить разворачивание (unrolling) тела цикла:
```C
if (a[i] != 0) {
    i++;
    while (a[i] != '\0') { i++ };
}
```
Условие продолжения цикла `a[i] != '\0'` пусть будет $$A(i)$$. Теперь, используя формулы для слабейшего предусловия (полученные Дейкстрой), запишем

$$Q(i) = (A(i) \Rightarrow wp(i++; S, R)) \land (\lnot A(i) \Rightarrow wp(skip, R))$$

Сначала упростим выражение

$$wp(i++; S, R) = wp(i++, wp(S, R)) = wp(i := i + 1, Q(i)) = Q(i + 1)$$

а также учтем, что $$wp(skip, R) = R$$. Что дает нам выражение

$$Q(i) = (A(i) \Rightarrow Q(i + 1)) \land (\lnot A(i) \Rightarrow R)$$

и далее упрощаем его до вида (с учетом того, что $$X \Rightarrow Y \equiv \lnot X \lor Y$$)

$$Q(i) = (\lnot A(i) \lor Q(i + 1)) \land (A(i) \lor R)$$

Инвариант цикла $$S$$ получаем при условии, что он не обрывается, что соответствует $$R = false$$. Таким образом, вечный цикл выглядит как

$$Q(i) = A(i) \land Q(i + 1)$$

и соответствует условию $$\forall i \geq 0$$, $$A(i)$$ (при условии, что стартовали мы с индекса $$i = 0$$). Или в исходных обозначениях $$\forall i \geq 0$$, `a[i] != '\0'`. Условия выхода из цикла (а тем самым и слабейшее предусловие) мы получаем, осуществляя инверсию: $$\exists i \geq 0$$, `a[i] == '\0'`.
